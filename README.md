# About

I design and implement retrieval-centered AI systems that combine Information Retrieval and Large Language Models.
My work focuses on domain-specific search and conversational AI systems, particularly in healthcare and other professional environments, where grounding, reliability, and efficiency are critical.

## Open source and development workflow

- I use GitHub primarily to open source research artifacts, prototypes, and reproducibility code.
- I run CI locally during development. Public repositories include minimal CI when it improves reproducibility.
- Several projects are maintained as private repositories while research is in progress or constrained by collaboration and institutional requirements.

## Public repositories and organisations

### 🔎 ASPIRE (open source tool)
**Assistive System for Performance Evaluation in Information Retrieval**  
Repo: https://github.com/GiorgosPeikos/ASPIRE  
Project page: https://giorgospeikos.github.io/projects/aspire-ir-evaluation/

A visual analytics platform for IR evaluation.

Demonstrates:

- Query-level analysis and diagnostics

- Evaluation beyond aggregate metrics
