# About

I design and implement retrieval-centered AI systems that combine Information Retrieval, Large Language Models.
My work focuses on building domain-specific search and conversational AI systems, particularly in healthcare and other professional environments, where grounding, reliability, and efficiency are critical.

# Selected Public Projects

## 🔎 ASPIRE  
**Assistive System for Performance Evaluation in Information Retrieval**

Repository:  
https://github.com/GiorgosPeikos/ASPIRE

A visual analytics platform for IR evaluation.

Demonstrates:

- Model comparison workflows

Frameworks:
Python, Streamlit, Pandas, IR metric pipelines
